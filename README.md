# Individuell uppgift – Hälsostudie


## Miljö
- **Python:** 3.13.7
- **Paket:** `Pandas`, `Numpy`, `Matplotlib`, `Scipy`, `statsmodels` (se `requirements.txt`)

## Kom igång
```bash
# klona projektet
git clone https://github.com/NadineKn/Python1.git

# Skapa och aktivera virtuell moljö
python -m venv .venv

# installera beroenden
python -m pip install -r requirements.txt

```
